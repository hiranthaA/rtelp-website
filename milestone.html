Milestones

Project milestones are defined as the achievement of specific activities of the project.

Project Proposal Report

Date: 12th March 2019
Marks Allocated: 5%

Project Proposal Report is the initial document that demarcate the initial requirements of the project activities. The proposal document describes all the techniques to be used in the research. Marks for the proposal document were 5 out of 100. Proposal Documents was supposed to submit on or before 12th of March 2019.

Project Proposal Presentation

Date: 15th March 2019
Marks Allocated: 5%

Proposal Presentation was conducted to evaluate the research idea and methodology for the proposed solution. On the 15th of March, 2019 .

SRS Document Submission

Date: 22nd April 2019
Marks Allocated: 6%

Importance of the Software Requirements Specification (SRS) document is to provide all the necessary information which are related to the requirements of the related system. Detailed description of what the system is about and how to use the system will be included in the SRS document for all the users who are willing to interact with the system.

Progress Presentation – I (50%)

Date: 19th June 2019
Marks Allocated: 10%

Progress Presentation – 1 was an assessment which gives a fresh, unbiased view of the project. Half of the project work supposed to be completed at the time of the presentation. This Presentation helps to identify potential for improvement, provides realistic and results-oriented recommendations.

Research Paper

Date: 15th July 2019
Marks Allocated: 10%

This document presents our evaluation or interpretation or argument. We build upon what we know about subject and make a attempt to find out what experts know.

Progress Presentation – II (90%)

Date: 3rd September 2019
Marks Allocated: 18%

Progress Presentation – 2 was mainly conducted to evaluate the completion of the research work of each component of the students. At least 90% of the research work should be done prior to this presentation. Completion of all the components along with the integration would be the ideal situation.

Final Report (Soft Bound)

Date: 16th September 2019
Marks Allocated: 15%

Final report is the final document concluding the project.

Website Assessment

Date: 21st September 2019
Marks Allocated: 4%

The website contains all the details about the project including documents and presentations associated with the project.

Final Presentation

Date: Yet to be done
Marks Allocated: 10%

All the research components of the project have to be completed and integrated prior to Final Assessment. Completed product has to be tested and finally should evaluate the accuracy results.

VIVA

Date: Yet to be done
Marks Allocated: 15%

Viva will be conducted on the same day in which final assessment will be evaluated. Questions will be asked from the students individually regarding their work on the research component in order to clarify their own contribution towards the components.
